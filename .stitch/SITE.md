# Bacaro Budget Manager — Site Vision & Roadmap
**Stitch Project ID:** 5004376411867209368

## 1. Vision
Bacaro Budget Manager is a premium personal finance dashboard application. It helps users track income, expenses, transfers, wallets, and savings goals. The app features an AI assistant called "Kwarta AI" for financial insights. The target audience is young professionals in the Philippines who want a modern, trustworthy financial tool.

## 2. App Pages (Screens Needed)
- **Landing Page** — Public marketing page to attract sign-ups.
- **Dashboard** — The main hub: balance overview, stat cards, cash flow chart, expense breakdown, recent transactions.
- **Transactions** — Full transaction list with search, filters, and sorting.
- **Wallets** — Grid of wallet cards with balance, type, and status. Add/transfer modals.
- **Goals** — Savings goals with progress bars, contribution history, and target tracking.
- **Settings** — Appearance (dark mode, contrast, language, currency), data export, legal links, danger zone.
- **Login / Sign Up** — Auth modals with floating labels, password toggle, forgot password flow.
- **Kwarta AI** — AI chat interface for financial insights and advice.

## 3. Tech Stack
- Frontend: React + Vite + Tailwind CSS + Shadcn/UI
- Backend: Node.js (Vercel Serverless) + Neon PostgreSQL + Drizzle ORM + Zod + TypeScript
